# Frontend - ML Interview Agent

Minimalistic Apple-style interface for the ML Interview Agent.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

## Features

- Clean, minimalistic Apple-style design
- Drag-and-drop resume upload
- Real-time upload progress
- Resume data extraction visualization
- Responsive design

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Axios for API calls
